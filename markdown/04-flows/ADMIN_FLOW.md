# Admin Flow

## Admin Login
1. Admin signs in using Google.
2. App checks admin permission.
3. If admin, redirect to dashboard.
4. If not admin, block access.

## Product Management
1. Admin opens Products page.
2. Admin can add new product.
3. Admin uploads product images.
4. Admin sets price, stock, category, and active status.
5. Admin can edit existing product.
6. Admin can deactivate or delete product.

## Order Management
1. Admin opens Orders page.
2. Admin sees recent orders.
3. Admin opens order detail page.
4. Admin can update order status.
5. Admin can see payment status.
